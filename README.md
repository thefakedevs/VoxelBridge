# VoxelBridge

VoxelBridge is a client-side mod for NeoForge and Fabric that exports in-game blocks, fluids, entities/block entities, and modded blocks to the glTF 2.0 format while preserving biome tinting.

The exported files can be used in DCC software such as Blender or other rendering tools.

## Features

*   Exports blocks, fluids, and entities/block entities.
*   Exports biome tinting (grass, foliage, water, and other vegetation).
*   Supports individual textures or UDIM packed atlases (up to 8192x8192 pixels per tile).
*   Adjustable atlas size and export thread count.
*   Supports most modded blocks, resource packs, and custom models (via resource packs).
*   Supports model and texture randomization (e.g., random offsets for grass, resource pack texture variants).
*   Supports material animation (exports frame sequences)..
